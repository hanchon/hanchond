# Addresses converter

Convert between Cosmos and Ethereum transactions

## Hex to Bech32

```sh
hanchond convert addr 0x6ddf27e021cbf9658be71e7966b44bb608810efe
evmos1dh0j0cppe0uktzl8reukddztkcygzrh7t5fx0n

hanchond convert addr 0x6ddf27e021cbf9658be71e7966b44bb608810efe -p osmo
osmo1dh0j0cppe0uktzl8reukddztkcygzrh7pwtcrf
```

## Bech32 to Hex

```sh
hanchond convert addr evmos1dh0j0cppe0uktzl8reukddztkcygzrh7t5fx0n
0x6DDf27E021CBf9658Be71E7966b44BB608810EFE

hanchond convert addr osmo1dh0j0cppe0uktzl8reukddztkcygzrh7pwtcrf
0x6DDf27E021CBf9658Be71E7966b44BB608810EFE
```
