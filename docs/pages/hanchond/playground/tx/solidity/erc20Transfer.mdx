# Transfer ERC20 coins

It executes an ERC20 transfer using the validator wallet.

Parameters:

- Contract Address
- Destination Address
- Token Amount

```sh
hanchond playground tx solidity erc20-transfer 0x13abd22cb7d1636a92c997bab22b99c7e9c85083 0xF45Ebe00d47a941776fAa42C5bc819166A75775C 1
{"txhash":"0x2a1d79fa17b252cf96fdb031bd3d3000770f64a6d9711b5871321fbdb0062434"}
```

:::info
The destination wallet can be in bech32 or hex format
:::
