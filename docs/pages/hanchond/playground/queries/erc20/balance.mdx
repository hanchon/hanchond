# Get Balance ERC20

The command `erc20 balance [contract] [wallet]` can be used to get the token balance of the given wallet.

```sh
hanchond playground query erc20 balance 0x80b5a32E4F032B2a058b4F29EC95EEfEEB87aDcd evmos1x49lse4ykqrvntuancrdjdjzz75xzmsj5nn9p0
5
```

:::info
The command supports wallets in `Bech32` and `Hex` encoding

The `height` flag is available to query archive data, it defaults to `latest`

The flag `url` can be used to change the URL provider.

The flag `mainnet` can be used to query Evmos Mainnet data. (This flag overwrites the `node` and `url` flag with the default Mainnet URL)
:::
